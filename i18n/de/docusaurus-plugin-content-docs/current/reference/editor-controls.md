# Bedienelemente im Anleitungseditor

Diese Referenz beschreibt die Funktionen im Anleitungseditor. Viele Bedienelemente sind **kontextabhängig** und erscheinen nur, wenn sie für die aktuelle Auswahl gelten.

## Werkzeugleiste

Über dem Maschenraster.

| Bedienelement | Aktion |
|---|---|
| **Neu** | Eine neue Anleitung beginnen. Bis zur Anmeldung ist die Schaltfläche deaktiviert. |
| **Form** | Den Dialog **Form einfügen** öffnen und eine [Form erzeugen](../explanations/materializing-shapes.md). |
| **Rückgängig** / **Wiederholen** | Bearbeitungsschritte zurücknehmen oder erneut anwenden. |
| Speicherstatus | Zeigt während der Bearbeitung **Ungesichert** und nach dem Speichern **Gespeichert**. |
| **Dateien** | Gespeicherte Anleitungen im Dateidialog öffnen und verwalten. |

## Kopf des Anleitungsbereichs

Über dem Maschenraster auf der linken Seite.

| Bedienelement | Aktion |
|---|---|
| Statusanzeige | Zeigt die aktuelle Arbeitsweise und die aktive Runde. |
| Chip **Maschenprobe** | Öffnet bei einer Projektanleitung die Projekteinstellungen, andernfalls den Dialog **Maschenprobe**. |
| Chip **Standard** | Öffnet den Dialog für die [Standardfarbe](../explanations/the-color-model.md). Das Farbfeld zeigt die aktuelle Farbe. |
| Vergrößern / verkleinern | Vergrößert den Anleitungsbereich auf die ganze Arbeitsfläche oder stellt die geteilte Ansicht wieder her. |

## Vorschau

Am unteren Rand der 3D-Vorschau auf der rechten Seite.

| Bedienelement | Aktion |
|---|---|
| **Realistisch** / **Schematisch** | Wechselt zwischen einer garnähnlichen Darstellung und dem Maschendiagramm. Nur die Vorschau ändert sich, nicht die Anleitung. |
| Vergrößern / verkleinern | Vergrößert die Vorschau auf die ganze Arbeitsfläche oder stellt die geteilte Ansicht wieder her. |

## Ein Häkelstück beginnen

Bei einer leeren Anleitung bietet das Raster drei Anfänge an (siehe [Arbeitsweise](./glossary.md)):

| Schaltfläche | Beginn |
|---|---|
| **Fadenring** | Ein geschlossener Ring als üblicher Amigurumi-Anfang. |
| **Offene Runde** | Ein offener Ring in der Mitte. |
| **Luftmaschenkette** | Eine Anfangskette, die in zwei Durchgängen behäkelt wird. |

## Maschenwerkzeuge

Unter dem Maschenraster. Die folgende Liste gilt für den normalen Bearbeitungsmodus. Während des Aufbaus eines Fadenrings oder einer Luftmaschenkette erscheinen eigene Werkzeuge.

### Kopfzeile

| Bedienelement | Aktion |
|---|---|
| Positionsanzeige | Zeigt für die aktuelle Auswahl die Koordinate `R# M#`. |
| Kontrollkästchen **Am Ende anfügen** | Neue Aktionen fügen Maschen am Ende des Häkelstücks hinzu. |

### Maschenaktionen

| Bedienelement | Aktion |
|---|---|
| Auswahl der Maschenart | `fM`, `hStb`, `Stb`, `DStb` oder `LM` für eine neue oder ausgewählte Masche wählen. Siehe [Maschenarten](./stitch-types.md). |
| **+ Masche** | Eine Masche am Ende anfügen. |
| **zun** | Eine [Zunahme](./stitch-types.md) anfügen. |
| **abn** | Eine [Abnahme](./stitch-types.md) anfügen. |
| **+ Basis** | Der ausgewählten Masche eine weitere Basismasche hinzufügen. |
| Masche löschen | Die zuletzt hinzugefügte Masche entfernen oder die aktuelle Runde wieder öffnen. |
| **Bereich auswählen** | Eine Bereichsauswahl bei der ausgewählten Masche beginnen. |
| Teilen | Die ausgewählte Masche in zwei Maschen teilen. |
| Zusammenführen ◀ / ▶ | Die ausgewählte Masche mit der linken oder rechten Nachbarmasche zusammenführen. |
| Einfügen ◀ / ▶ | Links oder rechts neben der ausgewählten Masche eine neue Masche einfügen. |
| Garnfarbe | Den Farbdialog für die ausgewählte Masche öffnen. |

Bei mehreren ausgewählten Maschen lässt sich nur ihre Maschenart gemeinsam ändern.

### Rundenaktionen

| Bedienelement | Aktion |
|---|---|
| Runde beenden | Die aktuelle Runde abschließen und die nächste beginnen. |
| **Auto-Aufbau** | Eine Zielzahl festlegen und die Runde passend aufbauen oder neu aufbauen. |
| Runde einfügen | An der aktuellen Position eine neue Runde einfügen. |
| Runde duplizieren | Die aktuelle Runde kopieren. |
| Runde löschen | Die ausgewählte Runde entfernen. |
| **Abschluss** | Den Dialog zum Schließen des Häkelstücks öffnen. |
| Fadenring / offene Runde | Die erste Runde zwischen geschlossenem Fadenring und offener Runde umschalten. |
