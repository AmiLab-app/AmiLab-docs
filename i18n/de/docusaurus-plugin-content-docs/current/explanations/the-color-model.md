# Das Farbmodell

AmiLab färbt ein Häkelstück mit zwei Ebenen und einer praktischen Abkürzung. Wenn du ihr Zusammenspiel kennst, kannst du eine ganze Form mit einem Klick umfärben und zugleich Details wie Augen oder Streifen auf einzelne Maschen setzen.

## Zwei Ebenen: Standardfarbe und eigene Maschenfarbe

Die **Standardfarbe** ist die Grundlage. Sie gilt für jede Masche ohne eigene Farbe. Wenn du sie änderst, wird das gesamte Häkelstück auf einmal umgefärbt. Ohne festgelegte Standardfarbe verwendet AmiLab einen neutralen Garnton.

Eine **Farbe pro Masche** überschreibt die Standardfarbe. Sobald du einer Masche eine eigene Farbe gibst, zeigt sie diese statt der Standardfarbe. So ergänzt du Details, ohne den Rest zu verändern: Einige gefärbte Maschen bilden ein Auge, eine ganze Reihe wird zu einem Streifen.

Die Regel lautet:

> Eine Masche zeigt ihre **eigene Farbe**, wenn eine festgelegt ist. Andernfalls zeigt sie die **Standardfarbe**.

Mit **Eigene Farbe löschen** fällt eine Masche auf die Standardfarbe zurück. Wenn du die Standardfarbe änderst, bleiben Maschen mit eigener Farbe unverändert.

## Die Abkürzung: Garnfarben des Projekts

Unter **Garnfarben** speicherst du eine kleine Farbauswahl für das Projekt. Sie färbt selbst nichts, sondern dient als Schnellzugriff. Sowohl der Dialog für die Standardfarbe als auch der Farbdialog einer einzelnen Masche zeigen diese Farben als direkt auswählbare Felder.

Dadurch bleiben die Teile eines Projekts konsistent: Jede Anleitung greift auf dieselben gespeicherten Farben zu.

## Zusammenspiel

Eine typische Figur verwendet alle drei Möglichkeiten:

1. Lege die **Standardfarbe** auf das Garn für den Körper fest – das ganze Teil wird aktualisiert.
2. Speichere Körper- und Kontrastfarbe unter **Garnfarben**, damit du sie schnell wiederverwenden kannst.
3. Gib einigen Maschen die Kontrastfarbe als **eigene Farbe**, um Augen oder Markierungen zu setzen.
