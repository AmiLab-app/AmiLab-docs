# Häkelanleitung als Text kopieren

AmiLab schreibt deine Anleitung während des Entwerfens automatisch als Text aus. Diesen Text können andere Runde für Runde nachhäkeln. Kopiere ihn, wenn du die Anleitung drucken, an anderer Stelle speichern oder versenden möchtest.

## Text kopieren

1. Öffne die Anleitung im Editor.
2. Suche unter dem Maschenraster den Bereich mit dem Runde-für-Runde-Text.
3. Klicke oben in diesem Bereich auf **Anleitungstext kopieren**.
4. Füge den Text in ein Dokument, eine Notiz-App oder eine Nachricht ein.

Der Text liegt nun in der Zwischenablage. AmiLab erzeugt ihn nach jeder Bearbeitung automatisch neu. Die kopierte Fassung entspricht daher immer dem aktuellen Häkelstück; du musst nichts aktualisieren oder erneut exportieren.

## Hinweise

- Der Anleitungstext berücksichtigt Maschen, Zu- und Abnahmen sowie einen angewendeten [Abschluss](./close-up-a-piece.md).
- Die Maschenzahlen stammen aus der Anleitung. Die reale Größe hängt von deiner [Maschenprobe](../explanations/gauge-and-dimensions.md) ab.
