# 简介

**AmiLab** 是一款基于浏览器的工具，用于设计带有实时3D预览的玩偶钩针编织图案。描述你的形状——球体、管状或自定义轮廓——AmiLab将生成逐圈针法说明，供你直接使用。

## 什么是玩偶？

玩偶（あみぐるみ）是日本用钩针编织小型填充动物和角色的艺术。形状从魔法圈开始，在连续的圈中逐针编织而成。

## AmiLab能做什么？

- **形状编辑器** — 绘制定义3D形状的轮廓曲线
- **实时3D预览** — 编辑时实时渲染形状
- **图案输出** — 以你的语言和符号生成逐圈针数
- **多语言** — 支持美式英语、英式英语、德语、荷兰语、日语、葡萄牙语和中文
