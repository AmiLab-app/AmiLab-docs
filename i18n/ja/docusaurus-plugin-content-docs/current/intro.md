# はじめに

**AmiLab**は、ライブ3Dプレビュー付きのあみぐるみかぎ編みパターンをデザインするためのブラウザベースのツールです。球体、チューブ、またはカスタムプロファイルなど、形状を指定するだけで、AmiLabが段ごとの目数指示を生成します。

## あみぐるみとは？

あみぐるみは、小さなぬいぐるみの動物やキャラクターをかぎ編みする日本の伝統的な技法です。輪の作り目から始めて連続する段で形状を作り上げていきます。

## AmiLabでできること

- **シェイプエディター** — 3D形状を定義するプロファイルカーブを描く
- **ライブ3Dプレビュー** — 編集しながらリアルタイムで形状をレンダリング
- **パターン出力** — お使いの言語と表記法で段ごとの目数を生成
- **多言語対応** — 米国英語、英国英語、ドイツ語、オランダ語、日本語、ポルトガル語、中国語に対応
