# 形状プロフィールエディターの操作項目

**形状プロフィール**画面では、再利用できる[プロフィール](../explanations/profiles-and-cross-sections.md)を作成します。メニュー（**☰**）の**画面 → 形状プロフィール**から開きます。

## ツールバー

| 操作項目 | 動作 |
|---|---|
| **新規** | 新しいプロフィールを作り、名前を付けます。 |
| 保存状態 | 編集中は**未保存**、保存後は**保存済み**と表示します。 |
| **ファイル** | 保存したプロフィールを開き、管理します。 |

## 設定項目

キャンバスの上にあります。

| 項目 | 動作 |
|---|---|
| **ゲージ** | ゲージ設定を開きます。3Dプレビューと生成時の目数に影響します。 |
| **断面** | クリックするたびに円形 → 円形＋ → 六角形 → 正方形の順で切り替えます。 |
| 寸法（`r:… h:…`） | **寸法**画面を開き、最大半径と全高を設定します。 |

## 断面

| 選択肢 | 結果 |
|---|---|
| **円形** | 滑らかな丸い断面。 |
| **円形＋** | 丸い断面で、段ごとに目の位置をずらします。 |
| **六角形** | 6面の断面。 |
| **正方形** | 4面の断面。 |

## 寸法

| 項目 | 範囲 |
|---|---|
| **最大半径** | 0.5〜50 cm |
| **全高** | 1〜100 cm |

## キャンバス

制御点を結んだ輪郭を表示する編集領域です。

| 操作 | 方法 |
|---|---|
| 点を選択 | 点をクリックします。 |
| 点を移動 | 点をドラッグします。上下端の点は半径方向だけに動き、中間の点は隣の点を越えない範囲で自由に動きます。 |
| **+ 点を追加** | 輪郭上で点の間隔が最も広い場所に制御点を追加します。 |
| **- 点を削除** | 選択中の点を削除します。上下端以外の点で使用できます。 |

全体の高さは、最上部の点を動かすのではなく、**寸法**画面の**全高**で変更します。
