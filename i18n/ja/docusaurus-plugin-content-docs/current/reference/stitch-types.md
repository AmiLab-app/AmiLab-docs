# 編み目の種類

AmiLabの日本語表示では、日本のかぎ針編みで一般的な名称を短く表示します。編み目の種類を選ぶ場所では、次の5種類を使用できます。

## 編み目

| 表記 | 名称 | 説明 |
|---|---|---|
| `細` | 細編み | 最も低い基本の編み目。あみぐるみで中心的に使います。 |
| `中長` | 中長編み | `細`より少し高さがあります。 |
| `長` | 長編み | 中長編みよりさらに高さがあります。 |
| `長々` | 長々編み | 対応する編み目の中で最も高さがあります。 |
| `鎖` | 鎖編み | 作り目の鎖や立ち上がりに使います。土台を持たない鎖編みは立ち上がりの鎖です。 |

表の下へ進むほど、`細`から`長々`まで編み目の高さが増します。高い編み目は1段で進む縦方向の距離が大きくなり、編み地の見た目と[パーツの寸法](../explanations/gauge-and-dimensions.md)に影響します。

## 増目と減目

`増目`と`減目`はエディターの操作として表示されますが、**独立した編み目の種類ではありません**。1段下の目との接続で決まります（[AmiLabがパーツを表現する仕組み](../explanations/how-amilab-models-a-piece.md)を参照）。

| 操作 | 表記 | 内容 |
|---|---|---|
| 増目 | `増目` | 同じ土台の目に2目編み入れ、段の目数を1つ増やします。 |
| 減目 | `減目` | 2つ以上の土台の目を一度に編み、段の目数を減らします。 |

増目と減目には5種類のどの編み目も使えます。編み目の種類を選び、増目または減目の操作で接続を設定します。
