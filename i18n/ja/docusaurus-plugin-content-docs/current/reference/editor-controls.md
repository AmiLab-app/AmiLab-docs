# 編み図エディターの操作項目

編み図エディターで使う操作項目の一覧です。多くの項目は**状況に応じて表示**され、現在の選択に使える場合だけ現れます。

## ツールバー

編み目グリッドの上にあります。

| 操作項目 | 動作 |
|---|---|
| **新規** | 新しい編み図を作ります。ログインするまで使用できません。 |
| **形状を挿入** | **形状を作成**画面を開き、[形状から編み図を生成](../explanations/materializing-shapes.md)します。 |
| **元に戻す** / **やり直す** | 編集操作を前後に移動します。 |
| 保存状態 | 編集中は**未保存**、保存後は**保存済み**と表示します。 |
| **ファイル** | 保存した編み図を開き、管理します。 |

## 編み図パネル上部

編み目グリッド左側の上部にあります。

| 操作項目 | 動作 |
|---|---|
| 状態表示 | 現在の作業モードと選択中の段を表示します。 |
| **ゲージ** | プロジェクト内の編み図ではプロジェクト設定を、プロジェクトなしではゲージ設定を開きます。 |
| **基本色** | [基本の毛糸色](../explanations/the-color-model.md)を設定します。色見本には現在の基本色が表示されます。 |
| 拡大 / 元に戻す | 編み図パネルを作業領域全体に広げるか、分割表示へ戻します。 |

## 3Dプレビュー

右側の3Dプレビュー下部にあります。

| 操作項目 | 動作 |
|---|---|
| **リアル表示** / **模式表示** | 毛糸らしい表示と編み目構造の表示を切り替えます。編み図自体は変わりません。 |
| **カメラ自動フォーカスをオン** / **カメラ自動フォーカスをオフ** | プレビューカメラを選択中の目へ移動するかを切り替えます。オンのときは、目を選択または追加するとカメラがその目へ移動します。オフのときは、選択や編集を続けてもカメラは動きません。 |
| 拡大 / 元に戻す | プレビューを作業領域全体に広げるか、分割表示へ戻します。 |

## パーツを編み始める

空の編み図では、次の3つの始め方を選べます（[作業モード](./glossary.md)を参照）。

| ボタン | 始め方 |
|---|---|
| **わの作り目** | 中心を閉じた輪。一般的なあみぐるみの始め方です。 |
| **開いた輪** | 中心を開けた輪。 |
| **鎖編みの作り目** | 作り目の鎖を2周に分けて編みます。 |

## 編み目編集ツール

編み目グリッドの下にあります。以下は通常の編集モードの項目です。わの作り目と鎖編みの作り目には専用の編集ツールがあります。

### 上部

| 操作項目 | 動作 |
|---|---|
| 位置表示 | 選択中の位置を`段# 目#`で示します。 |
| **末尾に追加** | オンにすると、操作した編み目をパーツの最後に追加します。 |

### 編み目の操作

| 操作項目 | 動作 |
|---|---|
| 編み目の種類 | 新しい目または選択中の目を`細`、`中長`、`長`、`長々`、`鎖`から選びます。[編み目の種類](./stitch-types.md)を参照してください。 |
| **+ 編み目** | 末尾に選択中の種類の目を追加します。 |
| **増目** | [増目](./stitch-types.md)を追加します。 |
| **減目** | [減目](./stitch-types.md)を追加します。 |
| **+ 土台目** | 選択中の目に土台を追加します。 |
| 編み目を削除 | 最後の目を削除するか、閉じた段を再び開きます。 |
| **範囲を選択** | 選択中の目から範囲選択を始めます。 |
| 分割 | 複数の土台を持つ目を個別の目に分けます。 |
| 結合 ◀ / ▶ | 選択中の目を左または右の目と結合します。 |
| 挿入 ◀ / ▶ | 選択中の目の左または右に新しい目を挿入します。 |
| 毛糸色 | 選択中の目に独自の色を設定します。 |

複数の目を同時に選択した場合は、編み目の種類だけをまとめて変更できます。

### 段の操作

| 操作項目 | 動作 |
|---|---|
| 次の段 | 現在の段を終え、次の段を始めます。 |
| **段の自動作成** | 目標の目数を指定し、その数に合うよう段を作成または再作成します。 |
| 段を挿入 | 現在の位置に新しい段を追加します。 |
| 段を複製 | 現在の段をコピーします。 |
| 段を削除 | 選択中の段を削除します。 |
| **閉じ方** | 編み終わり方を設定する画面を開きます。 |
| わを閉じる / わを開く | 最初の段を、わの作り目と開いた輪の間で切り替えます。 |
