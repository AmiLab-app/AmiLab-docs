# 形状とパラメーター

**形状を作成**画面で選べる形状と入力値の一覧です。生成の仕組みは[形状から編み図を生成する仕組み](../explanations/materializing-shapes.md)を参照してください。

## 形状

| 形状 | サイズ | 2つ目の寸法 | 説明 |
|---|---|---|---|
| **滑らかな球体** | 半径（cm） | 高さ（cm）、任意 | 高さを空欄にすると半径から自動計算します。 |
| **六角形の球体** | 半径（cm） | 高さ（cm）、任意 | 六角形の断面に目をずらして配置し、均一な球体を作ります。高さは空欄の場合に自動計算します。 |
| **ずらし配置の球体** | 半径（cm） | 高さ（cm）、任意 | 段ごとに目をずらして配置します。高さは空欄の場合に自動計算します。 |
| **筒** | 直径（cm） | 長さ（cm） | 指定した長さのまっすぐな円筒です。 |
| **平らな円** | 半径（cm） | — | 2つ目の寸法がない平らな円盤です。 |
| **保存したプロフィール** | 最大半径（cm） | 高さ（cm）、任意 | 保存した[プロフィール](./glossary.md)から生成します。最初はプロフィール自身の寸法が入ります。 |

## 入力範囲

| パラメーター | 範囲 | 刻み |
|---|---|---|
| サイズ（半径 / 直径） | 1〜30 cm | 0.5 |
| 高さ / 長さ | 0.5〜200 cm | 0.5 |

指定した大きさから生成される目数は[ゲージ](../explanations/gauge-and-dimensions.md)によって変わります。現在のゲージを使って、センチメートルを目数と段数へ換算します。
