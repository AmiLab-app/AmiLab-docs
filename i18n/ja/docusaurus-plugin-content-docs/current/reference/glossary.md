# 用語集

AmiLabとこのドキュメントで使う用語の短い説明です。詳しい解説がある場合はリンクを示します。

### 基本色

独自の色がないすべての目に使われる毛糸色です。変更するとパーツ全体の色が変わります。[配色の仕組み](../explanations/the-color-model.md)を参照してください。

### ゲージ

編み地10cmに入る目数と段数です。AmiLabはゲージを使って目数を実寸へ換算します。[ゲージと寸法](../explanations/gauge-and-dimensions.md)を参照してください。

### 減目（`減目`）

2つ以上の土台の目を一度に編み、段の目数を減らす編み方です。

### 作業モード

わの作り目、開いた輪、鎖編みの作り目など、パーツの始め方と組み立て方です。最初の編み目が決まります。

### 生成

形状と少数のパラメーターから、完成した編み図を作ることです。[形状から編み図を生成する仕組み](../explanations/materializing-shapes.md)を参照してください。

### 制御点

[プロフィール](#プロフィール)の輪郭上にあり、指定した高さでの太さ、つまり半径を決める点です。[プロフィールと断面](../explanations/profiles-and-cross-sections.md)を参照してください。

### 組み立て

完成した複数のパーツを配置し、1つのキャラクターや物としてまとめたものです。

### 増目（`増目`）

同じ土台の目に2目編み入れ、段の目数を1つ増やす編み方です。

### 断面

AmiLabがプロフィールの輪郭を立体化するときに使う形です。円形、円形＋、六角形、正方形があり、3D形状を決めます。[プロフィールと断面](../explanations/profiles-and-cross-sections.md)を参照してください。

### 段

パーツの周囲を一周する編み目です。AmiLabでは段を直接保存せず、編み目の接続から*求めます*。

### 土台の目

編み目を編み入れる1段下の目、または複数の目です。土台がない目からパーツが始まり、土台が2つ以上ある目は減目です。[AmiLabがパーツを表現する仕組み](../explanations/how-amilab-models-a-piece.md)を参照してください。

### とじ方

パーツを仕上げるために目を引き寄せる仮想の糸です。球体の上部を引き絞る糸などが該当します。

### 編み図

エディターで設計する1つのかぎ針編みパーツです。編み目と段から構成されます。

### プロジェクト

同じゲージと毛糸の配色を共有する、関連した編み図のグループです。

### プロフィール

**形状プロフィール**画面で作成する再利用可能な輪郭です。[制御点](#制御点)と[断面](#断面)で定義し、3D形状として展開して編み図を生成できます。[プロフィールと断面](../explanations/profiles-and-cross-sections.md)を参照してください。

### 毛糸の配色

プロジェクトに保存する少数の色です。色選択画面からワンクリックで再利用できます。[配色の仕組み](../explanations/the-color-model.md)を参照してください。
