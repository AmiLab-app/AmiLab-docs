# 配色の仕組み

AmiLabでは、2段階の色設定と再利用用の配色を組み合わせます。仕組みを理解すると、形状全体を一度に塗り替えながら、目や模様などの細部だけに別の色を付けられます。

## 基本色と目ごとの色

**基本色**は配色の土台です。独自の色がないすべての目に適用されるため、基本色を変えるだけでパーツ全体の色が変わります。基本色を設定していない場合は、標準の毛糸色が使われます。

各目には**個別の色**を設定できます。色を指定した目は基本色を使わず、独自の色を表示します。ほかの部分を変えずに細部を追加する方法です。数目なら目になり、一周分ならしま模様になります。

規則は次のとおりです。

> 独自の色がある目は**その色**を表示し、ない目は**基本色**を表示します。

**個別の色指定を解除**すると、その目は基本色に戻ります。基本色を変更しても、独自の色がある目は変わりません。

## プロジェクトの配色

**毛糸の配色**は、プロジェクトに保存する少数の色です。それ自体で編み目に色を付けるのではなく、色をすぐ選ぶために使います。基本色と目ごとの色のどちらの画面にも色見本として表示されます。

同じプロジェクトのすべての編み図から同じ色を選べるため、作品全体の配色をそろえられます。

## 組み合わせ方

一般的なキャラクターでは、次のように使います。

1. 体に使う毛糸を**基本色**に設定し、パーツ全体を塗り替えます。
2. 体の色と差し色を**毛糸の配色**に保存します。
3. 数目だけ差し色で**個別指定**し、目や模様を作ります。
