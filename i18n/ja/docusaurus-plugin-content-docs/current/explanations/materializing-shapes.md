# 形状から編み図を生成する仕組み

球体の目をすべて手作業で配置するには時間がかかります。**形状の生成**を使うと、少数の数値を指定するだけで、AmiLabが完成した編集可能な編み図を作ります。

## 「生成」とは

生成した形状は特別なデータではなく、通常の編み図です。AmiLabが現在のゲージに合わせて必要な目と段を計算し、手作業で作った場合と同じようにエディターへ配置します。その後は、ほかの編み図と同じように編集、配色、変形できます。

## パラメーターから編み目へ

形状の**サイズ**と、必要な場合は2つ目の寸法を入力します。AmiLabは次の処理を行います。

- [ゲージ](./gauge-and-dimensions.md)から、指定した長さに必要な目数と段数を求めます。
- 各段の目数を計算し、選んだ輪郭に沿うようにします。
- 編み目、土台との接続、必要な閉じ方を生成します。球体上部の絞り止めもここに含まれます。

計算にはゲージを使うため、同じ完成サイズでもゲージによって目数が変わります。固定の目数ではなく、指定した実寸を優先します。

使用できる形状、入力値、範囲は[形状とパラメーター](../reference/shapes-and-parameters.md)で確認できます。

## 現在の編み図が置き換わる

生成した形状はエディター全体を使用するため、**現在開いている編み図を置き換え**、元に戻す履歴も消去します。すでに目がある場合は確認画面が表示されます。空の編み図ではすぐに生成されます。

形状の生成は挿入ではなく出発点です。最初に基本形を作り、その後で編集や配色を行って目的のパーツに仕上げます。

## 保存したプロフィール

球体、筒、平らな円などの標準形状に加えて、**形状プロフィール**画面で作成した輪郭からも生成できます。AmiLabが輪郭を立体化するため、オリジナル形状を再利用できるテンプレートとして保存できます。
