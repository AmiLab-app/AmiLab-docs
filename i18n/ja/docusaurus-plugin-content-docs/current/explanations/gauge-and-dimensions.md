# ゲージと寸法

編み図は目数と段数で表しますが、完成した作品の大きさはセンチメートルで測ります。**ゲージはこの2つをつなぐ情報**であり、AmiLabが完成サイズを判断する基準です。

## ゲージが表すもの

ゲージは編み地の密度を表す2つの数値です。

- **10cmあたりの目数** — 横10cmに入る目の数。
- **10cmあたりの段数** — 高さ10cmに入る段の数。

値は毛糸、かぎ針、編む強さによって変わります。最も確実なのは、実際に編んだゲージ用の編み地を測る方法です。毛糸ラベルの値は目安として使えます。AmiLabの初期値は**10cmあたり18目、20段**です。目数は**1〜60**、段数は**1〜80**の範囲で設定できます。

## 寸法の計算方法

AmiLabはゲージと編み目グラフを組み合わせ、各目を3D空間に配置して全体を測ります。3Dプレビューの下に表示される**幅、高さ、奥行き**は、*現在のゲージで編んだ場合*の完成サイズです。

同じ目数でもゲージが違えば大きさは変わります。6段の球体でも、細いコットン糸では小さく、太い毛糸では大きくなります。編み図は同じでも、ゲージと完成品の大きさが異なります。

## ゲージを変えると全体が変わる理由

ゲージは1目ごとの設定ではなく、パーツ全体に適用されます。どちらかの値を変えるとAmiLabが配置を計算し直し、全体が拡大または縮小します。目数は変わらず、実寸だけが変化します。

[形状から編み図を生成](./materializing-shapes.md)するときにもゲージを使用します。指定した大きさの球体や筒を作るために必要な目数を、現在のゲージから計算します。

## ゲージの保存先

- **プロジェクト内** — ゲージはプロジェクトに属し、その中の編み図で共有されます。編み目グリッド上部の**ゲージ**をクリックすると、プロジェクト設定から変更できます。
- **プロジェクトなし** — ゲージはその編み図だけに保存されます。**ゲージ**をクリックすると、専用の設定画面が開きます。
