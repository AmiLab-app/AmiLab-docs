# プロフィールと断面

**形状プロフィール**画面では、標準一覧にない形を作れます。考え方は単純です。**形の片側の輪郭を描き、それを軸の周りに展開します。** 結果を決めるのは、*プロフィール*と*断面*です。

## プロフィール：制御点で作る輪郭

**プロフィール**は形状の半分の輪郭で、複数の**制御点**からできています。各点は「この高さでどのくらい太いか」を表します。

- 点の**高さ**は、パーツの下端からの距離をセンチメートルで表します。
- 点の**半径**は、その高さで中心軸からどこまで外側へ広がるかを表します。AmiLabでは、軸上を0、最大半径を1とする割合で保存するため、形状全体を1つの設定で拡大縮小できます。

点を軸へ近づけるとその高さで細くなり、外側へ動かすと膨らみます。細かく調整したい場所には点を追加し、不要な場所では削除します。点を結んだ線がプロフィールです。

輪郭を実寸にする設定は2つあります。

- **最大半径** — 形状が最も太くなる部分。半径の割合1がこの実寸に対応します。
- **全高** — 最上部の点の高さ。輪郭全体を縦方向に伸縮します。

## 断面：輪郭を立体化する方法

プロフィールが示すのは側面だけです。**断面**は形状を水平に切ったときの形で、AmiLabが輪郭をどのように立体化するかを決めます。

- **円形** — 滑らかな丸い形。
- **円形＋** — 丸い形で、段ごとに目の位置をずらします。
- **六角形** — 6面の形。
- **正方形** — 4面の形。

同じプロフィールでも断面によって印象が変わります。しずく形の輪郭は、円形なら滑らかな卵形になり、六角形なら面のある宝石のような形になります。

## プロフィールから編み図へ

プロフィールは再利用できる*テンプレート*であり、それだけでは編み図ではありません。[形状を生成](./materializing-shapes.md)するとき、AmiLabは輪郭と断面を[ゲージ](./gauge-and-dimensions.md)に組み合わせ、各段の目数を計算して編み目を作ります。標準形状と同じ処理です。プロフィールは保存されたままなので、別のサイズやゲージでも再利用できます。
