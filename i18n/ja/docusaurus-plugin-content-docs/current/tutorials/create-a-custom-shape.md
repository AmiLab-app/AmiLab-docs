# オリジナル形状を作る

標準の球体や筒で多くの形を作れますが、一覧にない形が必要になることもあります。**形状プロフィール**画面では、形の片側の輪郭である*プロフィール*を描き、3D形状として展開して編み図にできます。

このチュートリアルでは、簡単な**卵形**を作り、編み図エディターで生成します。

学習する内容：

- **形状プロフィール**画面を開き、新しいプロフィールを作る
- 形状の大きさと断面を設定する
- 制御点を移動、追加、削除して輪郭を作る
- オリジナル形状を編集可能な編み図として生成する

**所要時間：** 約15分。

**始める前に：** プロフィールを保存できるようログインしてください。エディターの基本操作を使用するため、先に[はじめに](./getting-started.md)を進めておくことを推奨します。

---

## ステップ1：形状プロフィール画面を開く

1. 右上のメニューボタン（**☰**）をクリックします。
2. **画面**の**形状プロフィール**をクリックします。

片側に編集キャンバス、反対側に3Dプレビューがあるプロフィールエディターへ切り替わります。

:::note[プロフィールと形状]

画面名は**形状プロフィール**で、そこで作る再利用可能な輪郭を**プロフィール**と呼びます。その輪郭を立体化した結果が形状です。

:::

---

## ステップ2：新しいプロフィールを作る

1. ツールバーの**新規**をクリックします。
2. **新しいプロフィールの名前**画面で`卵`などの名前を入力します。
3. **作成**をクリックします。

プロフィールがアカウントへ保存され、その後は編み図と同じように**自動保存**されます。保存状態は編集中に**未保存**、保存後に**保存済み**と表示されます。

---

## ステップ3：キャンバスの見方を知る

キャンバスには、複数の**制御点**を結んだ曲線としてプロフィールが表示されます。

- キャンバスの**左端**が形状の中心軸です。
- 縦方向が**高さ**、横方向が中心からの**半径**です。
- 各制御点は、**その高さで形状がどのくらい太いか**を決めます。軸へ近づけると細くなり、外側へ動かすと膨らみます。

新しいプロフィールは、上下に1点ずつある先細りの筒から始まります。キャンバス横の3Dプレビューに現在の輪郭からできる形が表示され、編集のたびに更新されます。

---

## ステップ4：大きさと断面を設定する

キャンバス上部の2つの項目で全体の形を設定します。

**寸法** — `r:… h:…`と表示される項目をクリックします。

1. **最大半径**を`4` cmに設定します。
2. **全高**を`7` cmに設定します。
3. **完了**をクリックします。

**断面** — 輪郭をどのように立体化するかを設定します。クリックするたびに次の選択肢へ切り替わります。

| 断面 | 結果 |
|---|---|
| **円形** | 滑らかな丸い断面。 |
| **円形＋** | 丸い断面で、目の位置を段ごとにずらします。 |
| **六角形** | 6面の断面。 |
| **正方形** | 4面の断面。 |

丸い卵形にするため、**円形**のままにします。

---

## ステップ5：卵形を作る

卵形は上下が丸く、中央より少し下が最も太くなります。

1. **下側を丸くする。** 一番下の点を左の軸へ近づけ、底を細くします。
2. **膨らみを作る。** **+ 点を追加**をクリックします。新しい点をキャンバス下側3分の1付近へ動かし、最大半径に近い右側まで広げます。
3. **上側を丸くする。** 一番上の点を軸へ近づけ、先端を柔らかく閉じます。
4. **曲線を滑らかにする。** 膨らみと上下端の間に1〜2点を追加し、卵らしい滑らかな線になるよう調整します。不要な点は選択して**- 点を削除**をクリックします。

変更するたびに3Dプレビューが更新されます。立体を確認しながら調整してください。

:::tip[制御点を動かせる範囲]

**上下端の点**は高さが固定され、左右方向、つまり半径だけを変更できます。全体の高さは**寸法**画面の**全高**で設定します。**中間の点**は、隣の点の間で自由に動かせます。

:::

輪郭が卵形になり、保存状態が**保存済み**になったら完成です。

---

## ステップ6：形状から編み図を生成する

プロフィールは再利用可能なテンプレートです。標準形状と同じように、編み図として生成すると実際に編める形になります。

1. メニュー（**☰**）を開き、**画面**の**編み図**をクリックします。
2. ツールバーの**新規**から空の編み図を作り、`わたしの卵`などの名前を付けて**作成**します。
3. ツールバーの**形状を挿入**をクリックして**形状を作成**画面を開きます。
4. **形状**を開き、**保存したプロフィール**から**卵**を選びます。
5. プロフィールの**サイズ**と**高さ**が入力されます。そのまま使うか、完成サイズを変更します。
6. **この形状を生成**をクリックします。

編み目グリッドに卵を作る段が追加され、3Dプレビューに表示されます。以降は通常の編み図として、配色、編み目の編集、編み終わりの設定ができます。

:::note[ゲージと編み図の置き換え]

AmiLabは現在の[ゲージ](../explanations/gauge-and-dimensions.md)から目数を計算するため、完成した卵は設定した実寸になります。形状は編み図全体を使うので、すでに目がある場合は置き換えの確認が表示されます。今回は空の編み図から始めています。

:::

---

## 完成したもの

オリジナルの卵形を作り、かぎ針編みの編み図へ変換しました。プロフィール**卵**は保存した形状として残り、別のサイズやゲージでも繰り返し生成できます。
