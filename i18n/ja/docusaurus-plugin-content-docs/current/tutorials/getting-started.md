# はじめに

このチュートリアルでは、アカウントへのログインから色付きの形状を完成させるまで、AmiLabで最初のデザインを作ります。最後には、好きな毛糸色と2つの「目」を付けた六角形の球体が編み図として保存されます。多くのあみぐるみキャラクターの基本になる形です。

学習する内容：

- ログインして作業をアカウントへ保存する
- 編み図を作り、プロジェクトへまとめる
- ゲージと毛糸の配色を設定する
- 六角形の球体を生成する
- 全体の色を変え、目になる色のまとまりを追加する

**所要時間：** 約15分。

**必要なもの：** Googleアカウントと最新のデスクトップブラウザー。横幅の広い画面では、編み図とプレビューの分割表示を確認しやすくなります。

---

## ステップ1：ログインする

編み図に名前を付けて保存するにはアカウントが必要です。最初にログインします。

1. ブラウザーでAmiLabを開きます。
2. 画面右上のメニューボタン（**☰**）をクリックします。
3. **アカウント**で**Googleでログイン**をクリックします。
4. Googleのログイン画面で操作を完了します。

ログインすると、**アカウント**に名前が表示されます。メニューの外側をクリックすると閉じます。

:::note[最初にログインする理由]

アカウントがなくてもエディターを試せますが、**新規**は使用できず、変更は永続的に保存されません。ログインすると、保存、プロジェクト、自分のファイルを使用できます。

:::

---

## ステップ2：編み図とプロジェクトを作る

*編み図*は、かぎ針編みで作る1つのパーツです。*プロジェクト*は関連する編み図をまとめ、ゲージと毛糸の配色を共有します。同じプロジェクトのパーツを同じ条件で作れます。

1. 編み目グリッド上部のツールバーで**新規**をクリックします。
2. **新しい編み図の名前**画面で**新規プロジェクト**をクリックします。
3. **新規プロジェクト**画面に入力します。
   - **名前** — 例：`はじめてのあみぐるみ`
   - **ゲージ** — 編み地の密度を表す2つの数値。次のステップで説明します。
   - **毛糸の配色** — デザインで繰り返し使う色。ステップ4で設定します。
4. **作成**をクリックします。**新しい編み図の名前**画面へ戻り、新しいプロジェクトが選択された状態になります。
5. 編み図名に`ボール`などと入力し、**作成**をクリックします。

名前が付いた空の編み図がエディターで開きます。

:::note[作業は自動保存されます]

編み図を作成した時点でアカウントへ保存され、その後の変更も自動保存されます。保存状態は編集中に**未保存**、保存完了後に**保存済み**と表示されます。タブを閉じても、後で**ファイル**から開けます。

:::

:::tip[プロジェクトを使わない場合]

プロジェクトを作らず、名前だけを入力して**作成**することもできます。操作は同じですが、ゲージと色を編み図ごとに設定します。

:::

---

## ステップ3：ゲージを設定する

*ゲージ*は、編み地10cmに入る目数と段数です。AmiLabはこの値から実寸を計算し、使用する毛糸とかぎ針に合う3Dプレビューと完成サイズを表示します。

**新規プロジェクト**画面の**ゲージ**には、2つの入力欄があります。

- **目**の欄に**10cmあたりの目数**を入力します。例：`18`
- **段/10cm**の欄に**10cmあたりの段数**を入力します。例：`20`

毛糸ラベルの値を目安にするか、実際に編んだゲージ用の編み地を測ってください。

:::note[後でゲージを変える]

プロジェクト内の編み図では、ゲージはプロジェクトに保存されます。編み目グリッド上部の**ゲージ**をクリックすると、プロジェクト設定を開いて変更できます。プロジェクトなしの編み図では、専用の**ゲージ**画面が開きます。変更するとパーツ全体の実寸が変わります。

:::

---

## ステップ4：毛糸の配色を追加する

毛糸の配色には、後で繰り返し使う色を保存します。毎回カラーピッカーで探さず、色見本から選べます。

**新規プロジェクト**画面の**毛糸の配色**で次を行います。

1. **色を追加**をクリックします。ランダムな色の見本が追加されます。
2. 色見本をクリックし、カラーピッカーで色を選びます。
3. 同じ操作を繰り返します。このチュートリアルでは、**体の色**と目に使う**濃い差し色**を追加します。

色を削除するには、色見本の横にあるごみ箱をクリックします。

設定後、プロジェクト作成時は**作成**、既存プロジェクトの編集時は**保存**をクリックします。

---

## ステップ5：六角形の球体を生成する

すべての目を手動で配置する代わりに、形状から編み図を**生成**できます。少数の入力値から、編集可能な完成した編み図が作られます。六角形の球体では、六角形の断面に目をずらして配置し、均一なボールを作ります。

1. ツールバーの**形状を挿入**をクリックします。
2. **形状を作成**画面の**形状**で**六角形の球体**を選びます。
3. **サイズ**に球体の半径をセンチメートルで入力します。`4`を試してください。
4. **高さ**は空欄にし、自動計算させます。
5. **この形状を生成**をクリックします。

編み目グリッドに各段が追加され、右側の3Dプレビューに球体が表示されます。プレビューをドラッグすると回転できます。

:::note[現在の作業を置き換える]

今回は空の編み図なので、すぐに球体が表示されます。すでに目がある場合は確認画面が表示されます。形状を生成すると現在の編み図と元に戻す履歴が置き換わるためです。

:::

---

## ステップ6：リアル表示に切り替えて体の色を選ぶ

*基本色*は、独自の色を設定していないすべての目に使う毛糸色です。基本色を変えると球体全体の色が一度に変わります。

まず、毛糸としての見え方を確認できるリアル表示へ切り替えます。

1. 3Dプレビュー下部の**リアル表示**をクリックします。編み目の模式図から、毛糸らしい表示へ変わります。ボタンは**模式表示**に変わり、クリックすると構造表示へ戻ります。
2. 編み目グリッド上部の**基本色**をクリックします。
3. **基本の毛糸色**画面で次のどちらかを選びます。
   - **プロジェクトの配色**から色見本をクリックする
   - 大きい色見本をクリックし、カラーピッカーで色を選ぶ
4. **完了**をクリックします。

球体全体が選んだ体の色に変わります。

:::tip[模式表示とリアル表示]

**模式表示**は各目を記号で示し、構造の確認や個別の目の選択に適しています。**リアル表示**は毛糸に近い色と質感で、配色の確認に適しています。切り替えても編み図は変わりません。

:::

---

## ステップ7：色のまとまりで目を作る

差し色を付けた数目をまとめると、顔の目として見せられます。少し間を空けて2か所に色を付けます。

1. 編み目グリッドで球体中央付近の幅が広い段を探します。
2. 1目クリックして選択します。下部の編み目編集ツールが更新されます。
3. 色付きの四角で表示される**編み目の毛糸色**をクリックします。
4. **プロジェクトの配色**から濃い差し色を選び、**完了**をクリックします。
5. 隣の目にも同じ色を付け、2〜3目で1つ目の目を作ります。
6. 数目空けて、同じ大きさのまとまりをもう1つ作ります。

**リアル表示**では、色を付けた目が毛糸として表示されるため、顔の間隔を確認しながら調整できます。選びにくい目がある場合は**模式表示**へ切り替えてください。

:::tip[元に戻す、色を解除する]

ツールバーの**元に戻す**と**やり直す**で変更を調整できます。目を基本色へ戻すには、その目を選択して毛糸色の画面を開き、**個別の色指定を解除**をクリックします。

:::

---

## 完成したもの

プロジェクト内に、名前付きで保存された六角形の球体が完成しました。体の基本色、2つの目、再利用できる毛糸の配色とゲージが設定されています。同じプロジェクトに作る次のパーツでもゲージと配色を共有できます。
