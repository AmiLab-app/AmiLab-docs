# 編み図を文章としてコピーする

AmiLabは編み図を作成しながら、実際に編むときに使える段ごとの文章も生成します。印刷する、別の場所に保存する、ほかの人へ送るときにコピーしてください。

## テキストをコピーする

1. 編み図をエディターで開きます。
2. 編み目グリッドの下にある、段ごとの編み方を表示するパネルを探します。
3. パネル上部の**編み方テキストをコピー**をクリックします。
4. 文書、メモアプリ、メッセージなどへ貼り付けます。

テキストがクリップボードへコピーされます。AmiLabは編集のたびに文章を自動生成するため、コピーした内容は常に現在のパーツと一致します。更新や再出力の操作は必要ありません。

## 注意点

- 文章には編み目、増目、減目、設定した[編み終わり](./close-up-a-piece.md)が反映されます。
- 目数は編み図から決まり、実寸は[ゲージ](../explanations/gauge-and-dimensions.md)によって変わります。
