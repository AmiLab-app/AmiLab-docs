# 保存数の上限を管理する

無料アカウントでは、最大1件のプロジェクトと6件の編み図を保存できます。AmiLabは、アカウント内に保存されたプロジェクトと編み図を数えます。編み図の複製、共有された組み立てのリミックス、ログイン後に匿名の作業をアカウントへ移したものも含まれます。

上限に達すると、AmiLabは**無料プランの保存上限に達しました**を表示します。既存の作業はそのまま使えます。

## 無料プランの保存上限に達しましたに対応する

1. 画面の内容を読み、どちらの上限に達したか確認します：**プロジェクト**または**編み図**。
2. **料金プランを見る**をクリックして**料金**を開きます。
3. さらに保存したい場合は、アクセス期間を選びます。

料金には固定のアクセス期間があります：**31日間**と**12か月間**。これはアクセス期間であり、サブスクリプションではありません。

## 無料上限内で作業を続ける

有料アクセスなしで続けるには、次の操作を使います。

- **ファイル**から既存の編み図を開いて編集する。
- 不要になった保存済みの作業を名前変更、移動、削除する。
- 編み図の保存枠が残っている場合だけ、プロジェクトなしの編み図を作る。
- 保存済みプロジェクトがまだない場合だけ、新しいプロジェクトを作る。

編み図を複製すると、新しい保存済み編み図が1件作られます。そのため、6件の編み図上限に数えられます。

## 有料アクセスが期限切れになった場合

アクセス期限が切れても、既存の作業は開く、編集する、名前を変える、移動する、削除することができます。

新しいプロジェクトや編み図を作る場合は、次のアクセス期間を選ぶまで無料アカウントの上限が適用されます。アカウント内の保存数がすでに無料上限を超えている場合は、既存ファイルで作業を続けるか、**無料プランの保存上限に達しました**が表示されたときに**料金プランを見る**を使います。
