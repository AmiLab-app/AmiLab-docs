# 操作ガイド

AmiLabで特定の作業を完了するためのガイドです。目的に合うページを選び、手順に沿って操作してください。

- [編み図を文章としてコピーする](./copy-pattern-as-text.md) — 段ごとの編み方をテキストで取り出します。
- [1目ずつパーツを作る](./build-a-piece-by-hand.md) — 増目と減目を使いながら手動で形を作ります。
- [編み終わりを閉じる](./close-up-a-piece.md) — 絞り止めまたは平らにとじて開口部を仕上げます。
- [編み図をプロジェクトで整理する](./organize-patterns-into-projects.md) — 保存した編み図を開く、移動する、まとめる方法です。
- [保存数の上限を管理する](./manage-saved-work-limits.md) — 無料アカウントの上限とアクセス期間への対応を確認します。
- [パーツを組み立てる](./combine-pieces-into-a-composition.md) — 複数の編み図を1つの作品として配置します。
- [組み立てとプレビューを共有する](./share-a-composition.md) — 公開リンク、プレビュー画像、リミックスを設定します。
