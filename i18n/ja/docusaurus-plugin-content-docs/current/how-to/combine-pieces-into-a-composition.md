# パーツを組み立てる

完成したあみぐるみは、胴体、頭、腕、耳など、複数のパーツからできています。**組み立て**では、保存した編み図を1つの配置にまとめ、キャラクター全体を確認して共有できます。

## 始める前に

- ログインしてください。組み立てはアカウントに保存されます。
- 同じ**プロジェクト**に複数の編み図を保存してください。組み立ては1つのプロジェクトに属し、その中の編み図だけをパーツとして使います。[はじめに](../tutorials/getting-started.md)を参考に、先に編み図を作成します。

## 1. 組み立て画面を開く

1. メニュー（**☰**）を開きます。
2. **画面**の**組み立て**をクリックします。

## 2. 新しい組み立てを作る

1. ツールバーの**新規**をクリックします。
2. **新しい組み立ての名前**画面で所属する**プロジェクト**を選び、名前を入力します。
3. **作成**をクリックします。

追加できるパーツは、選択したプロジェクトの編み図です。

## 3. パーツを追加する

1. **パーツを追加**をクリックします。
2. 一覧から保存済みの編み図を選びます。**パーツ**パネルと3Dプレビューに追加されます。
3. キャラクターに必要なパーツをすべて追加します。

耳を2つ、脚を4つなど、同じ編み図を必要な回数だけ追加できます。

## 4. 各パーツの位置と向きを調整する

**パーツ**パネルで対象を選び、変形設定を使います。

| 操作項目 | 動作 | 範囲 |
|---|---|---|
| **位置** X / Y / Z | 空間内でパーツを移動します（cm）。 | −50〜50 |
| **回転** X / Y / Z | パーツを回転します（度）。 | −180〜180 |
| **左右反転** | 右用のパーツを左用にする場合などに反転します。 | オン / オフ |

値を変えるとプレビューがすぐに更新されます。**パーツ**パネルから直接反転または削除することもできます。

:::tip[左右対称のパーツを作る]

腕、耳、脚などの組は、同じ編み図を2回追加します。1つ目を配置し、2つ目を反対側へ移動して**左右反転**をオンにします。共有時には、反転したパーツの文章も自動的に逆向きになります。

:::

## 5. 全体を確認する

- プレビューをドラッグして全体を回転します。
- 毛糸らしい色で見る場合は**リアル表示**、構造を見る場合は**模式表示**を選びます。

組み立てはプロジェクトへ**自動保存**されます。保存した組み立ては**ファイル**から開いて管理できます。
