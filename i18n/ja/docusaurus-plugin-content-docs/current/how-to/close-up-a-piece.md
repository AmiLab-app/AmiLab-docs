# 編み終わりを閉じる

形を作り終えても、最後の段には開口部が残ります。**閉じ方**を設定すると、その部分の仕上げ方法が3Dプレビューと文章の編み図に反映されます。

## 始める前に

**閉じ方**は、パーツが2段以上あり、最終段に2目以上あり、その段を選択している場合に使えます。表示されないときは、段を追加するか最終段を選択してください。

## 閉じ方を設定する

1. 編み目編集ツールの**段**にある**閉じ方**をクリックします。
2. **編み終わりの設定**画面で方法を選びます。
   - **絞り止め** — 残りの目に糸端を通し、引き絞ります。球体など丸いパーツに適しています。
   - **平らにとじる** — 最終段を平らに折り、向かい合う端をまっすぐとじます。平底、袋、四角い形に適しています。
3. **平らにとじる**場合は、折り目の角になる目を選びます。その目から向かい合う目を組にしてとじます。
4. **適用**をクリックします。

操作項目が**設定済み ✓**に変わり、プレビューと文章に仕上がりが反映されます。

## 閉じ方を変更または解除する

もう一度**編み終わりの設定**を開くと方法を変更できます。**閉じ方を解除**をクリックすると開口部が元に戻ります。
