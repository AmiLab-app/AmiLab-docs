# 編み図をプロジェクトで整理する

**プロジェクト**は関連する編み図をまとめ、共通の[ゲージ](../explanations/gauge-and-dimensions.md)と[毛糸の配色](../explanations/the-color-model.md)を設定します。保存した編み図とプロジェクトは**プロジェクトと編み図**画面で管理します。

アカウントに保存した内容を扱うため、ログインが必要です。

## ファイル画面を開く

ツールバーの**ファイル**をクリックします。**プロジェクトと編み図**画面に、各プロジェクトのゲージと配色、その後にプロジェクトなしの編み図が表示されます。

## 編み図を操作する

各編み図には次の操作があります。

| 操作 | 動作 |
|---|---|
| **開く** | 編み図をエディターで開きます。 |
| 名前を変更（鉛筆） | 編み図に新しい名前を付けます。 |
| 複製（コピー） | 個別に編集できる複製を作ります。 |
| **移動** | 別のプロジェクトへ移動します。 |
| 削除（ごみ箱） | 確認後に編み図を削除します。 |

## プロジェクトを操作する

- **プロジェクトを作成** — **新規プロジェクト**をクリックし、名前、ゲージ、毛糸の配色を設定します。
- **編み図を追加** — プロジェクト内の**新しい編み図**を使うか、既存の編み図を**移動**します。
- **プロジェクトを編集** — 見出しの鉛筆をクリックし、名前、ゲージ、配色を変更します。設定はプロジェクト内のすべての編み図に適用されます。
- **プロジェクトを削除** — 見出しのごみ箱をクリックします。

## プロジェクトなしの編み図

プロジェクトを選ばずに作成した編み図は**プロジェクトなし**に表示されます。操作は同じですが、ゲージと色を独自に持ちます。**移動**を使うと、いつでもプロジェクトへ追加できます。
