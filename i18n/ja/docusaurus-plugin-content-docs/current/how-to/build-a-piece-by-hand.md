# 1目ずつパーツを作る

[形状から編み図を生成](../explanations/materializing-shapes.md)するとすぐに基本形を作れますが、1目ずつ手動で組み立てることもできます。標準形状では作れないパーツを設計するときや、構造を詳しく確認したいときに使います。

このガイドでは、ログインして新しい編み図を作成済みであることを前提にします。

## 1. 作り目を選ぶ

空の編み図で**どの作り目から始めますか？**と表示されたら、次から選びます。

- **わの作り目** — 中心を閉じた輪。一般的なあみぐるみの始め方です。
- **開いた輪** — 中心が開いた輪です。
- **鎖編みの作り目** — 鎖を作り、2周に分けて編みます。

一般的なあみぐるみでは**わの作り目**を選びます。

## 2. 最初の段を作る

**わの作り目**を選ぶと、専用の編集ツールへ切り替わります。

1. 編み目の種類を選びます。通常は`細`を使います。
2. **+ 編み目**をクリックし、輪へ1目ずつ追加します。確定するには最低限の目数が必要です。
3. 目数が決まったら**確定**をクリックします。

最初の段が確定し、通常の編集モードへ切り替わります。

## 3. 編み目と段を追加する

通常の編集では**末尾に追加**をオンにします。次の操作で編み図の最後に編み目を追加できます。

- **+ 編み目** — 選択中の種類を1目追加します。
- **次の段** — 現在の段を終え、次の段を始めます。編集ツールの**段**にある**↵アイコン**です。ツールチップには**この段を終えて次の段へ進む**と表示されます。

編み目を追加すると、段の目数と3Dプレビューがすぐに更新されます。

## 4. 増目と減目で形を作る

増目と減目を使うと、筒から球体や円すいの形を作れます。**末尾に追加**がオンの状態で次を使います。

- **増目** — 同じ土台の目に2目編み入れます。段の目数が1つ増え、パーツが太くなります。
- **減目** — 2つの土台の目を一度に編みます。段の目数が1つ減り、パーツが細くなります。

段全体を指定した目数にしたい場合は**段の自動作成**を使います。完成時の目数を入力すると、AmiLabが増目または減目を均等に配置して段を作成、または再作成します。

:::tip[目数を確認する]

各段の目数と3Dプレビューを見ながら形を調整してください。一定の割合で増目すると丸みができ、目数を保つとまっすぐな側面になります。最後に減目すると開口部が小さくなります。

:::
