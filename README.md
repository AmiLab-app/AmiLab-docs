# AmiLab Docs

Documentation for [AmiLab](https://github.com/AmiLab-app), a browser-based tool for designing amigurumi crochet patterns with a live 3D preview.

Built with [Docusaurus](https://docusaurus.io/). Supports 5 locales: English (US and UK), German, Dutch, and Japanese.

## Local development

```bash
npm install
npm run start                        # English (default)
npm run start -- --locale de         # German
npm run start -- --locale ja         # Japanese
```

## Build

```bash
npm run build        # builds all locales
npm run serve        # preview the production build
```
