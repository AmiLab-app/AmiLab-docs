# Design a Custom Character

Combine shapes and custom profiles to design an original amigurumi character.

## What You'll Learn

- How to use the Custom shape type
- How to combine multiple components

## Steps

1. **Create a new project** and select **Custom**
2. The profile editor shows a blank canvas. Click to add control points
3. Draw the cross-section of your desired shape — a pear, a star, an animal head
4. Use **Smooth** to round sharp transitions between points
5. The 3D preview updates in real time as you edit
6. Once the main body is done, repeat the process for smaller components (ears, limbs)
7. Save each component as a separate project
8. **Export** the pattern for each component

**Tip:** Start with a reference sketch next to your browser window. It helps to map out control point positions before you click.
