# Design a Tube

Tubes create arms, legs, tentacles, and cylindrical bodies.

## What You'll Learn

- How to create a tube shape
- How to taper the ends

## Steps

1. **Create a new project** and select **Tube**
2. The profile shows a rectangular outline. Drag the top and bottom control points to adjust the opening width
3. Use the middle control points to add taper — narrower at the wrist, wider at the shoulder
4. The **Height** slider changes the number of rounds
5. Review the pattern in the **Pattern** panel
6. **Export** your finished design

**Tip:** Combine a sphere (head) with a tube (body) to create a basic character silhouette.
