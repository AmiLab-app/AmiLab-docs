# Your First Amigurumi

Create a simple sphere — the building block of most amigurumi designs.

1. **Open AmiLab** and click **New Project**
2. In the shape selector, choose **Sphere**
3. The profile view shows a half-circle curve. Drag the control points to adjust the shape
4. The 3D preview updates live as you edit
5. Open the **Pattern** panel to see the round-by-round stitch instructions
6. Click **Export** to save your pattern

**Result:** A complete, crochetable pattern for a sphere.
