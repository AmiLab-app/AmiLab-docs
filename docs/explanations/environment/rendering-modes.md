# Rendering Modes

AmiLab provides two 3D rendering modes to suit different needs:

- **Solid** — the default view, showing your shape as a finished object with lighting and shading. Best for evaluating overall proportions and appearance.
- **Wireframe** — overlays the mesh structure on the shape. Best for understanding how rows and stitches are distributed.

Switch between modes in the **View** toolbar. The wireframe view is especially useful when fine-tuning the stitch distribution across the shape.
