# Localization System

AmiLab supports 7 stitch notation systems to match crochet terminology around the world.

Each locale uses a **terminology map** — a translation table that converts generic stitch instructions (increase, decrease, single crochet) into the region-specific abbreviation.

When you switch languages in **Settings**, the pattern panel re-renders using the selected map. The 3D preview and profile editor are unaffected — only the text output changes.

**Supported notation systems:** US English, UK English, German, Dutch, Japanese, Portuguese, and Chinese.
