# Export a Pattern

Download your finished pattern as a printable or shareable file.

1. Open your completed project in the **Editor** workspace
2. Click **Export** in the top toolbar
3. Choose a format:
   - **PDF** — formatted for printing
   - **TXT** — plain text, easy to paste into notes or messages
   - **JSON** — machine-readable, for sharing between AmiLab instances
4. The file downloads automatically

**Result:** A pattern file you can print, share, or archive.
