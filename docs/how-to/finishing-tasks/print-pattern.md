# Print a Pattern

Print your pattern directly from the browser for use at your hook.

1. Open your project in the **Editor** workspace
2. Click **Export → PDF**
3. Open the downloaded PDF
4. Press **Ctrl+P** (or **Cmd+P** on Mac) to open the print dialog
5. Select your printer and click **Print**

**Tip:** The PDF is formatted to fit a standard letter or A4 page. Adjust the print scale if needed.
