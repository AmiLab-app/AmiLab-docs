# Save and Organize

Keep your projects organized so you can find and continue them later.

1. Click **Save** in the top toolbar to save the current project to your browser's local storage
2. Open **My Projects** to see all saved designs
3. Click the name of a project to rename it
4. Use the **Search** bar to filter projects by name
5. Click **Delete** to remove a project

**Tip:** Projects saved in the browser are only available on that device. Use Sync to access them everywhere.
