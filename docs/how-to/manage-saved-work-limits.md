# Manage saved-work limits

Free accounts can save up to 1 project and 6 patterns. AmiLab counts saved projects and patterns in your account, including copies you create by duplicating a pattern, remixing a shared composition, or moving anonymous work into your account after sign-in.

When you reach a limit, AmiLab shows **Free limit reached**. Your existing work stays available.

## Respond to Free limit reached

1. Read the dialog to see which limit you reached: **projects** or **patterns**.
2. Click **View pricing** to open **Pricing**.
3. Choose an access window if you want to create more saved work.

Pricing offers fixed access windows: **31 days** and **12 months**. These are access periods, not subscriptions.

## Stay within the free limit

To keep working without paid access:

- Open and edit an existing pattern from **Files**.
- Rename, move, or delete saved work you no longer need.
- Create an ungrouped pattern only if you still have pattern capacity.
- Create a new project only if you do not already have one saved project.

Duplicating a pattern creates another saved pattern, so it counts toward the 6-pattern limit.

## If paid access expires

You can still open, edit, rename, move, and delete existing work after access expires.

Creating new projects or patterns returns to the free-account limits until you choose another access window. If your account already has more saved work than the free limit allows, keep working with existing files or use **View pricing** when **Free limit reached** appears.
