# Import a Profile

Use a profile curve exported from another tool or shared by another user.

1. Open the **Profile** workspace
2. Click **Import**
3. Select a supported file format (`.csv`, `.json`, or `.svg`)
4. The control points are placed automatically on the grid
5. Adjust any points as needed
6. Click **Save** to add the imported profile to your library

**Result:** The imported profile is ready to use in new projects.
