# Create a Custom Shape

Design a freeform profile for shapes that don't fit sphere or tube templates.

1. Open the **Profile** workspace
2. Click **New Profile** to start from a blank canvas
3. Click anywhere on the grid to place control points
4. Drag points to adjust the curve. The shape preview updates in real time
5. Use **Smooth** to round corners between points
6. Use **Mirror** to duplicate edits symmetrically
7. Name your profile and click **Save**

**Result:** The custom shape is saved to your profile library for reuse.
