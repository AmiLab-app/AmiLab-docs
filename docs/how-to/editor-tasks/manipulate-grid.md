# Manipulate the Grid

The grid view shows each round as a visual ring of stitches. Use it for fine-grained editing.

1. Open your project and switch to the **Grid** view
2. Each cell represents a stitch. Click a cell to select it
3. Use the toolbar to **increase** or **decrease** stitches in the selected round
4. Drag across multiple cells to bulk-select and apply changes
5. Changes reflect in both the 3D preview and the pattern text

**Result:** A visual, stitch-by-stitch way to shape your design.
