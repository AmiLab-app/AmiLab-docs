# Modify Rows

Add or remove rows from your pattern to change the height of your shape.

1. Open your project in the **Editor** workspace
2. In the **Pattern** panel, click **Add Row** to insert a new round at the end
3. Drag a row handle to reorder rounds
4. Click the **Delete** icon next to a row to remove it
5. The shape recalculates and the 3D preview updates

**Result:** The shape height changes to match the new row count.
