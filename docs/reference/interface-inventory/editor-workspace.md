# Editor Workspace

| Element | Description |
|---------|-------------|
| **Profile Editor** | Left panel. Draw and adjust the 2D cross-section curve |
| **3D Preview** | Center panel. Live rendered view of the shape |
| **Pattern Panel** | Right panel. Round-by-round stitch instructions |
| **Toolbar** | Top bar. Shape selector, undo/redo, save, export |
| **Settings** | Gear icon. Gauge, language, rendering options |
