# Toolbar Buttons

| Button | Action |
|--------|--------|
| **New** | Create a new project |
| **Save** | Save current project to local storage |
| **My Projects** | Open saved projects panel |
| **Undo** | Undo the last action |
| **Redo** | Redo a previously undone action |
| **Export** | Download pattern as PDF, TXT, or JSON |
| **Settings** | Open gauge, language, and display settings |
| **Sign In** | Sign in with Google for cloud sync |
