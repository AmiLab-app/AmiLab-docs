# Profile Workspace

| Element | Description |
|---------|-------------|
| **Canvas** | Large drawing area where you place and edit control points |
| **Tool Palette** | Pencil tool (add points), select tool (move points), eraser (remove) |
| **Smooth Toggle** | Enable to round sharp transitions between control points |
| **Mirror Toggle** | Enable to duplicate all edits symmetrically across the center axis |
| **Library** | Saved profiles you can reuse in any project |
