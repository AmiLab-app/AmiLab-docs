# Terminology

| Term | Definition |
|------|------------|
| **Magic ring** | A starting technique where stitches are worked into an adjustable loop |
| **Round** | One complete set of stitches around the circumference |
| **Increase (inc)** | Work two stitches into one stitch from the previous round |
| **Decrease (dec)** | Work two stitches together to reduce the count |
| **Gauge** | The number of stitches and rows per 10 cm with a given yarn and hook |
| **Profile** | The 2D cross-section curve that defines a 3D shape |
| **Control point** | A draggable point on the profile curve |
| **Spline** | A smooth curve interpolated between control points |
