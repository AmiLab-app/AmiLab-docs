# File Formats

## Export Formats

| Format | Extension | Use Case |
|--------|-----------|----------|
| PDF | `.pdf` | Printing, sharing as a formatted document |
| Plain Text | `.txt` | Pasting into notes, messages, or other apps |
| JSON | `.json` | Machine-readable transfer between AmiLab instances |

## Import Formats (Profiles)

| Format | Extension | Source |
|--------|-----------|--------|
| CSV | `.csv` | Spreadsheet or data export |
| JSON | `.json` | AmiLab profile export |
| SVG | `.svg` | Vector design tools |
