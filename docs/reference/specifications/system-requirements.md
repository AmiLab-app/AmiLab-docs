# System Requirements

AmiLab runs in the browser. No installation is required.

## Supported Browsers

- Chrome 90+
- Firefox 90+
- Safari 15+
- Edge 90+

## Hardware

- **GPU:** WebGL 2.0 support required for 3D preview
- **RAM:** 4 GB minimum
- **Screen:** 1024 × 768 minimum resolution

## Internet

- A network connection is required to load the app
- Once loaded, the app runs locally. Pattern generation does not require an active connection
- Cloud sync requires internet access
